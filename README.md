# Averqon - Startup Company Website

A premium, modern corporate website inspired by the HCL theme, built with React, Tailwind CSS, and Framer Motion.

## Features

- **Premium UI/UX**: Clean corporate style with modern gradients and glassmorphism.
- **Dynamic Hero Section**: Features typewriter effects, animated 3D-style illustrations, and glass dashboard mockups.
- **Interactive Components**: Hover-lift service cards, animated counter stats, and smooth scroll animations.
- **Dark Mode Support**: Seamless transition between light and dark themes.
- **Fully Responsive**: Mobile-first design with a sticky, scroll-adaptive navbar.
- **Performance Optimized**: Built with Vite and modern React practices.

## Tech Stack

- **Frontend**: React.js, Tailwind CSS
- **Animations**: Framer Motion, CountUp.js, React Type Animation
- **Icons**: Lucide React
- **Slider**: Swiper.js
- **Routing**: React Router Dom

## Getting Started

1. **Clone the repository** (if applicable)
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the development server**:
   ```bash
   npm run dev
   ```
4. **Build for production**:
   ```bash
   npm run build
   ```

## Folder Structure

```
src/
 ├── assets/          # Static assets
 ├── components/      # UI Components (Navbar, Hero, Services, etc.)
 │    └── Common/     # Reusable components (Preloader, FloatingWidgets)
 ├── pages/           # Page layouts (Home, etc.)
 ├── App.jsx          # Root component & Routing
 └── main.jsx         # Entry point
```
